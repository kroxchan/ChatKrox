# 6大投资哲学视角详解

本文件详细说明6种投资世界观的分析框架。在执行第三步时阅读本文件。

每种哲学不只是"不同的估值方法"——它是一整套看待世界的方式。同一组财报数据，在这6副"眼镜"下看起来可能完全不同。正是这种差异，帮助你避免只用一个视角思考的盲区。

---

## 1. 质量复利视角（Quality Compounder）

**哲学**："以合理价格买入绝妙的生意，远胜以绝妙价格买入平庸的生意。"

**代表人物**：巴菲特（Warren Buffett）、芒格（Charlie Munger）、李录、段永平

**时间范围**：永久持有。卖出是例外，不是常态。

**关键指标**：ROIC（投入资本回报率）的长期趋势

### 这个视角关注什么

- **护城河的持久性**：20年后这家公司的竞争优势还在吗？品牌、网络效应、成本优势、转换成本——哪些在变宽，哪些在被侵蚀？
- **资本配置能力**：管理层拿赚到的钱做了什么？再投资回报率高吗？分红和回购是否明智？
- **定价权**：公司能不能持续涨价而不流失客户？这是最简洁的护城河测试
- **ROIC趋势**：连续5年以上的ROIC走势，比单一年份更重要

### 这个视角如何评估这只股票

回答一个问题：**如果市场关闭10年，你拿着这只股票能安心睡觉吗？**

- 能 → 考虑以合理价格买入并长期持有
- 不能 → Pass（但可能适合其他视角的短期交易）

## 2. 想象力成长视角（Imaginative Growth）

**哲学**："最大的投资错误是想象力不足。"

**代表人物**：Baillie Gifford（Tom Slater、James Anderson）、Cathie Wood（ARK）、Sequoia

**时间范围**：5年以上，以十年为单位思考

**关键指标**：收入增长率、TAM（潜在总市场）的可实现性

### 这个视角关注什么

- **先花30分钟只讨论上行空间**：如果一切顺利，这家公司能有多大？
- **TAM远景**：市场系统性地低估变革性增长的持续时间和幅度
- **波动不是风险**：股价大幅回调不等于投资失败。Conviction要能扛住drawdown
- **非线性增长的可能性**：是否存在S曲线加速的拐点？

### 这个视角如何评估这只股票

回答一个问题：**5年后回头看，今天不买会不会后悔？**

- 如果TAM不够大、增长故事不够compelling → Pass
- 如果存在"想象力失败"的风险（市场远远低估了上行空间）→ Long，接受高波动

## 3. 基本面多空视角（Fundamental Long/Short — Tiger Cubs）

**哲学**："用深度研究买入最好的公司，做空最差的公司。"—— Julian Robertson

**代表人物**：Julian Robertson（起源）、Viking（Halvorsen）、Lone Pine（Mandel）、Coatue（Laffont）

**时间范围**：1-3年

**关键指标**：EV/EBITDA、Variant Perception（变异认知）

### 这个视角关注什么

- **Variant Perception**（变异认知）：100小时的深度研究能揭示什么市场漏掉的东西？
- **对冲思维**：不只是买赢家，还要做空输家——从两侧获利
- **快速止损**：如果错了，快速退出。不死扛
- **"What's priced in?"**：当前股价已经反映了什么预期？什么是市场还没定价的？

### 这个视角如何评估这只股票

回答一个问题：**有没有一个市场还没看到的Variant View？**

- 有 → 做多或做空，取决于Variant View的方向
- 没有（你的看法和市场一致）→ Pass，没有alpha

## 4. 深度价值视角（Deep Value）

**哲学**："我专门买别人在卖的、不受待见的、被嫌弃的东西。"

**代表人物**：Seth Klarman（Baupost）、Howard Marks（Oaktree）

**时间范围**：数年（耐心等待均值回归）

**关键指标**：重置成本（Replacement Cost）、安全边际（Margin of Safety）

### 这个视角关注什么

- **安全边际至上**：以折扣足够大的价格买入，即使分析有误也不会亏太多
- **"二级思维"**（Howard Marks）：当所有人都认为某样东西有风险时，这个共识本身通过压低价格**降低了**实际风险
- **私人买家标准**：如果有人想把整家公司买下来（包括承担所有负债），会出多少钱？如果市场价比这个数低40%以上，值得关注
- **不需要催化剂**：时间和均值回归会完成价值回归的工作

### 这个视角如何评估这只股票

回答一个问题：**市场出价比这家公司的"清算价值"低多少？**

对大部分高增长科技股，深度价值视角的结论通常是Pass——因为这些公司的价值主要在未来增长而非现有资产。但如果一只科技股因为市场恐慌被打到了接近净现金的价格，深度价值视角就变得很有价值。

## 5. 催化剂驱动视角（Event-Driven / Catalyst）

**哲学**：识别会触发市场重新定价的具体事件。

**代表人物**：David Tepper、John Paulson、Carl Icahn、Bill Ackman、Paul Singer（Elliott）

**时间范围**：6-18个月

**关键指标**：催化剂时间线、概率加权回报

### 这个视角关注什么

- **催化剂是edge的核心**：分拆、重组、破产退出、代理权争夺、监管变化
- **时间窗口**：机会是有期限的，需要在窗口内兑现
- **概率加权回报**：催化剂成功的概率 × 成功时的回报 - 催化剂失败的概率 × 失败时的损失
- **"如果说不出催化剂的名字，你就不是事件驱动——你只是在希望。"**

### 这个视角如何评估这只股票

回答一个问题：**未来6-18个月，有什么具体、可识别的事件会迫使市场重新给这只股票定价？**

- 有明确催化剂 + 概率加权回报吸引 → Long或Short
- 没有明确催化剂 → Pass

催化剂示例：新产品发布、大型客户合同、监管裁决、管理层变动、分拆/合并、财报beat/miss

## 6. 宏观战术视角（Macro-Tactical — Druckenmiller School）

**哲学**："推动整体市场的不是盈利，是美联储。"

**代表人物**：Stanley Druckenmiller、Scott Bessent

**时间范围**：随宏观周期而定

**关键指标**：美联储政策、流动性环境

### 这个视角关注什么

- **50/30/20框架**：一只股票50%的走势由大盘决定、30%由行业决定、只有20%是公司自身——如果宏观方向看对了，选股错误也能被原谅
- **流动性是镜头，美联储是太阳**：宏观政策体制决定了游戏场地
- **"预判别人还没预判到的变化"**：不是看当前宏观数据，而是看二阶变化——市场还没反映的政策转向
- **当确信时重仓**："对的时候当猪——贪婪地下注"

### 这个视角如何评估这只股票

回答一个问题：**当前的宏观/流动性环境对这只股票是顺风还是逆风？**

- 顺风 + 个股基本面好 → 放大仓位
- 逆风 + 个股基本面好 → 减小仓位或等待
- 逆风 + 个股基本面差 → 做空候选

如果用户已安装 `macro-liquidity` 或 `us-market-sentiment` skill，建议联动使用以获取最新宏观环境判断。

`fed policy outlook rate decision` `financial conditions index` `[代码] macro sensitivity beta` `liquidity cycle phase`

## 综合使用指南

6个视角分析完毕后：

1. **投票统计**：几个视角看多、几个看空、几个Pass？
2. **分歧分析**：如果视角之间有严重分歧，分歧的根源是什么？（通常是时间范围不同或关注点不同）
3. **权重调整**：
   - 你的投资期限更接近哪个视角，那个视角的权重就更高
   - 核心持仓候选 → 质量复利和想象力成长的权重更高
   - 战术持仓候选 → 基本面多空和催化剂驱动的权重更高
4. **Cross-Framework Acknowledgment**：即使最终结论是Pass，也要说明"另一种风格的投资者可能会怎么看"
